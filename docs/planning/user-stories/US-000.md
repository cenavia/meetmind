## User Story

**Como** Pablo (desarrollador) o Tech Lead  
**Quiero** una estructura de proyecto inicial con las tres capas definidas y un flujo mínimo end-to-end funcional  
**Para** validar el setup antes de implementar las funcionalidades completas y tener un punto de partida reproducible

---

## Descripción ampliada

Esta user story establece el esqueleto del proyecto MeetMind según la arquitectura de tres capas (API, Negocio, UI). El objetivo es tener un "Hello World" funcional donde:

1. Un usuario interactúa con la UI Gradio introduciendo texto
2. La UI envía la petición a la API FastAPI
3. La API invoca el grafo LangGraph
4. El grafo retorna datos mock estructurados (participantes, temas, acciones, minuta, resumen)
5. La UI muestra el resultado al usuario

Todo el flujo debe ser ejecutable siguiendo instrucciones documentadas en `COMO-EJECUTAR.md`.

---

## Criterios de Aceptación

- [ ] Estructura de directorios según [ARQUITECTURA](../../ARQUITECTURA-Sistema-Procesamiento-Reuniones.md): `src/api/`, `src/agents/meeting/`, `src/db/`, `src/services/`, `src/ui/`
- [ ] **Capa API (FastAPI):** `main.py` con `GET /health` y un endpoint `POST /api/v1/process/text` minimal que acepta JSON y devuelve respuesta mock
- [ ] **Capa Negocio (LangGraph):** Grafo mínimo con nodo preprocess + un nodo de prueba que retorna datos hardcodeados (participantes, temas, acciones, minuta, resumen en formato esperado)
- [ ] **Capa UI (Gradio):** App minimal con un input de texto, botón "Procesar" que invoca la API o el grafo directamente, y salida que muestra el resultado
- [ ] Flujo end-to-end: Usuario escribe texto en Gradio → se envía a API → API invoca grafo → grafo retorna estado → respuesta se muestra en UI
- [ ] Dependencias en `pyproject.toml` gestionadas con **uv**: FastAPI, Gradio, LangGraph, LangChain
- [ ] Documento `docs/planning/COMO-EJECUTAR.md` con instrucciones claras de ejecución

## Instrucciones de Ejecución (entregable)

El documento `COMO-EJECUTAR.md` debe incluir:

1. **Requisitos previos:** Python 3.11+, [uv](https://docs.astral.sh/uv/) instalado
2. **Instalación:**
   ```bash
   uv sync
   # o: uv pip install -e .
   ```
3. **Ejecutar la API:**
   ```bash
   uv run uvicorn src.api.main:app --reload --host 0.0.0.0 --port 8000
   ```
   - Verificar: `curl http://localhost:8000/health`
4. **Ejecutar la UI (Gradio):**
   ```bash
   uv run gradio src/ui/app.py
   # o: uv run python -m gradio src/ui/app.py
   ```
   - La UI debe poder invocar la API en `http://localhost:8000` (o el grafo directamente en modo demo)
5. **Ejecutar el grafo en modo standalone (opcional):**
   ```bash
   uv run python -c "
   from src.agents.meeting.agent import get_graph
   g = get_graph()
   result = g.invoke({'raw_text': 'Reunión de prueba con Juan y María.'})
   print(result)
   "
   ```
